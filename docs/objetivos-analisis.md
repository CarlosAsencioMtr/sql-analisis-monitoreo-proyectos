# Objetivos del Análisis

Este proyecto busca demostrar el uso de SQL para analizar información relacionada con el monitoreo de proyectos.

## Objetivos principales

- Calcular el total de proyectos por estado.
- Analizar el avance promedio por dirección.
- Identificar proyectos atrasados.
- Evaluar la carga de proyectos por responsable.
- Calcular la duración promedio estimada de proyectos.
- Generar KPIs útiles para un dashboard empresarial.

## Preguntas de negocio

1. ¿Cuántos proyectos están activos, finalizados, nuevos o atrasados?
2. ¿Qué dirección tiene mayor cantidad de proyectos?
3. ¿Cuál es el avance promedio por dirección?
4. ¿Qué responsables tienen más proyectos asignados?
5. ¿Qué proyectos presentan atraso?
6. ¿Cuál es el avance promedio general?
7. ¿Cuál es el presupuesto total de los proyectos?

## Uso esperado

Las consultas SQL pueden utilizarse como base para alimentar dashboards en Power BI, reportes ejecutivos o análisis de monitoreo de proyectos.
